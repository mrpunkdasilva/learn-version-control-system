Comandos Essenciais do Git

Cheat Sheet (Tabela de preguiçoso)

Essa tabela fornece uma visão geral dos principais comandos Git e suas funcionalidades básicas.

Comando Git

Descrição

git init

Inicializa um novo repositório Git

git add <arquivo>

Adiciona um arquivo modificado à área de stage

git add .

Adiciona todos os arquivos modificados à área de stage

git commit -m "Mensagem do commit"

Cria um novo commit com a mensagem especificada

git mv <arquivo-original> <arquivo-novo>

Renomeia ou move um arquivo no repositório
