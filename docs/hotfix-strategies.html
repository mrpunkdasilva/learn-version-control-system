Comandos Git para Hotfix

1. Workflow Git

# Criar hotfix branch git checkout -b hotfix/bug-123 main # Commit fix git commit -m "fix: corrige bug crítico #123" # Merge em main git checkout main git merge --no-ff hotfix/bug-123 # Sync develop git checkout develop git merge --no-ff hotfix/bug-123

2. Git Flow

# Iniciar hotfix git flow hotfix start bug-123 # Finalizar hotfix git flow hotfix finish bug-123
