1.1 O que os Sistemas Operacionais fazem

Um sistema computadorizado ou só computador, pode ser dividido em quatro partes:

  • Hardware

  • Sistema Operacional

  • Software

  • Usuários

  • Também podemos considerar que um sistema computadorizado é composto por:
